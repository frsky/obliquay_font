# Obliquay

Obliquay is a friendly, mildly off-kilter headline font with a touch of 60s movie poster. OTF and FontForge provided for Regular, Regular Bold, Condensed, and Condensed Bold.

### Charset Support:

  - Latin-1
  - Google Latin
  - Partial Latin Extended-A
